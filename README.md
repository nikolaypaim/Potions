# Potions

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

página web onde você escolhe uma entre três poções que adicionam uma mensagem diferente para cada opção.

---
## 📚 Aprendizados
- condicionais
- estilização
- funções

## ⚙️ Funcionalidades
- escolha uma poção e clique nela para receber uma mensagem de alerta
- existem três poções: uma vermelha (cura), uma azul (mana) e uma verde (veneno)

## 🛠️ Stack


- JavaScript
- HTML5
- CSS3

## 🚀 Deploy
1. Clone ou baixe este repositório.

```bash
git clone https://github.com/nikolaypaim/Potions
```

2. Abra o arquivo `index.html` em seu navegador.
